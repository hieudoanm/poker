# Blackjack

## Features

- Full **52-card deck** (♠, ♥, ♦, ♣)  
- **Hi-Lo card counting** system:  
  - 2–6 → +1  
  - 7–9 → 0  
  - 10–A → -1  
- Keyboard controls:  
  - **TAB** → Deal next card  
  - **SPACE** → Reveal running count  
  - **Q** → Reset deck  
- Stops automatically when all 52 cards are dealt  
- Styled with **TailwindCSS + DaisyUI**
